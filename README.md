# ФинКратко bot — subtitle-first free mode

Этот вариант не скачивает аудио с YouTube. Он работает так:
1. Находит новый ролик на канале.
2. Пытается получить русские субтитры.
3. Проверяет качество субтитров.
4. Если субтитры нормальные — делает конспект и публикует в Telegram.
5. Если субтитров нет или они плохие — кладет ролик в `data/needs_review.json`.

## Что нужно настроить в GitHub

Secrets:
- `YOUTUBE_API_KEY`
- `TELEGRAM_BOT_TOKEN`
- `TELEGRAM_CHAT_ID`
- `OWN_CHANNEL_URL`

Variable:
- `CHANNEL_HANDLES` — список каналов через запятую.

## Важные файлы
- `data/pending_queue.json` — отложенные до следующего окна публикации посты.
- `data/needs_review.json` — ролики, где субтитров нет или они плохие.
- `debug_reports/*.json` — отчеты по каждому обработанному ролику.

## Окно публикации
По умолчанию: с 08:00 до 21:00 МСК.
Если ролик вышел после 21:00 — пост уходит в `pending_queue.json` и будет отправлен на следующем дневном запуске.
